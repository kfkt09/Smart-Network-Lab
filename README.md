# Smart-Network-Lab
This project involves the design and implementation of a computer network using Cisco Packet Tracer. The goal is to create a secure and well-structured network that connects multiple devices and departments within a virtual organization. Routers, switches, and VLAN technology were used to organize network traffic and improve performance. Basic networking protocols were also applied to ensure efficient communication between all devices. This project helped me develop my skills in network design, IT infrastructure management, and understanding how real-world networks operate.
